## 2026-01-07 - [Accessibility] The Missing Link
**Learning:** Even well-structured semantic HTML can fail keyboard users if navigation is repetitive. A "Skip to Main Content" link is a critical, often overlooked pattern that dramatically improves efficiency for keyboard and screen reader users.
**Action:** Always check for a skip link as the first item in the body for any page with a header/nav menu.
